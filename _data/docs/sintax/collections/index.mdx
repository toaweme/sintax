## Collections

| Modifier | Description |
|----------|-------------|
| [`filter`](./filter) | Filter returns a subset of a slice where a nested field matches a value. |
| [`find`](./find) | Find returns the first element in a slice or map where a field equals the given value. |
| [`first`](./first) | First returns the first character of a string or the first element of a slice. |
| [`flatten`](./flatten) | Flatten flattens a slice of slices by one level. |
| [`has`](./has) | Has returns true if the slice or map contains the given value. |
| [`is`](./is) | Is returns true if the value equals any of the given parameters. |
| [`key`](./key) | Key extracts a value from a map or slice by key path or index. |
| [`last`](./last) | Last returns the last character of a string or the last element of a slice. |
| [`map`](./map) | Map converts a slice of maps into a map keyed by the given field's string value. |
| [`merge`](./merge) | Merge converts a slice of maps into a map keyed by the given field's string value. |
| [`pluck`](./pluck) | Pluck extracts a single field from each element of a slice of maps and returns a slice of values. |
| [`sort`](./sort) | Sort sorts a slice in ascending or descending order. |
| [`sum`](./sum) | Sum returns the numeric sum of the elements of a slice. |
| [`wrap`](./wrap) | Wrap wraps the value in a map under the given key. |
