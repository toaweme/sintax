## Convert

| Modifier | Description |
|----------|-------------|
| [`from`](./from) | From parses the string value as the given format and returns the parsed result. |
| [`json`](./json) | JSON serializes the value to a JSON string. |
| [`markdown`](./markdown) | Markdown converts an HTML string to Markdown. |
| [`yaml`](./yaml) | YAML serializes or parses a value as YAML. |
