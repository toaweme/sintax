From parses the string value as the given format and returns the parsed result.

## Input

`string` 

## Parameters

| # | Type | Required |
|---|------|----------|
| 0 | `string ("json" | "csv")` | yes |

## Returns

`map for json, []map[string]any for csv`

## Examples

### Parse a JSON string into a map

**Input**

```
body = "{\"name\": \"Alice\", \"role\": \"admin\"}"
```

**Template**

```
{{ body | from:'json' }}
```

**Output**

```
{"name": "Alice", "role": "admin"}
```

### Parse a CSV string into a list of rows

**Input**

```
body = "name,age\nAlice,30\nBob,25"
```

**Template**

```
{{ body | from:'csv' }}
```

**Output**

```
[{"name": "Alice", "age": "30"}, {"name": "Bob", "age": "25"}]
```
