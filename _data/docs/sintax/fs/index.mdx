## File System

| Modifier | Description |
|----------|-------------|
| [`dirname`](./dirname) | Dirname returns the directory portion of a file path. |
| [`ext`](./ext) | FilenameExt returns the file extension without the leading dot. |
| [`ext-dot`](./ext-dot) | FilenameExtDot returns the file extension including the leading dot. |
| [`ext-prepend`](./ext-prepend) | FilenamePrependExt inserts an additional extension before the existing file extension. |
| [`ext-trim`](./ext-trim) | FilenameTrimExt returns the file path without its extension. |
| [`filename`](./filename) | Filename returns the base file name from a path, including the extension. |
