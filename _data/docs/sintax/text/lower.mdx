ToLower converts a string to lowercase.

## Input

`string` 

## Parameters

None

## Returns

`string`

## Examples

### Normalize an email address

**Input**

```
email = "Alice@Example.COM"
```

**Template**

```
{{ email | lower }}
```

**Output**

```
alice@example.com
```

### Lowercase a heading

**Input**

```
title = "Hello, World!"
```

**Template**

```
{{ title | lower }}
```

**Output**

```
hello, world!
```
