Concat appends one or more strings to the value.

## Input

`string` 

## Parameters

| # | Type | Required |
|---|------|----------|
| ... | `string` | yes |

## Returns

`string`

## Examples

### Append a punctuation mark

**Input**

```
greeting = "Hello"
```

**Template**

```
{{ greeting | concat:'!' }}
```

**Output**

```
Hello!
```

### Build a kebab-cased identifier

**Input**

```
prefix = "user"
suffix = "profile"
```

**Template**

```
{{ prefix | concat:'-' | concat:suffix }}
```

**Output**

```
user-profile
```
