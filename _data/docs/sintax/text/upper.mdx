ToUpper converts a string to uppercase.

## Input

`string` 

## Parameters

None

## Returns

`string`

## Examples

### Shout a name

**Input**

```
name = "Alice"
```

**Template**

```
{{ name | upper }}
```

**Output**

```
ALICE
```

### Format a country code

**Input**

```
code = "us"
```

**Template**

```
{{ code | upper }}
```

**Output**

```
US
```
