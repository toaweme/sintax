Reverse reverses the characters in a string.

## Input

`string` 

## Parameters

None

## Returns

`string`

## Examples

### Reverse a name

**Input**

```
name = "Alice"
```

**Template**

```
{{ name | reverse }}
```

**Output**

```
ecilA
```

### Reverse a short code

**Input**

```
code = "ABC123"
```

**Template**

```
{{ code | reverse }}
```

**Output**

```
321CBA
```
