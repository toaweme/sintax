title: "gt"
description: "Gt returns true if the numeric value is greater than the threshold."
category: boolean
---

Gt returns true if the numeric value is greater than the threshold.

## Input

`int` `float` 

## Parameters

| # | Type | Required |
|---|------|----------|
| 0 | `number` | yes |

## Returns

`bool`

## Examples

### Check if a basket has items

**Input**

```
items_in_cart = 3
```

**Template**

```
{{ items_in_cart | gt:0 }}
```

**Output**

```
true
```

### Check if a price exceeds free-shipping threshold

**Input**

```
total = 49.99
```

**Template**

```
{{ total | gt:50 }}
```

**Output**

```
false
```
