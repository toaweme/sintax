Not inverts the truthiness of the value.

## Input

`any` 

## Parameters

None

## Returns

`bool`

## Examples

### Invert an account status

**Input**

```
is_active = true
```

**Template**

```
{{ is_active | not }}
```

**Output**

```
false
```

### Detect a clean run with no errors

**Input**

```
errors = []
```

**Template**

```
{{ errors | not }}
```

**Output**

```
true
```
