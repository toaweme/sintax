---
title: "gte"
description: "Gte returns true if the numeric value is greater than or equal to the threshold."
category: boolean
---

Gte returns true if the numeric value is greater than or equal to the threshold.

## Input

`int` `float` 

## Parameters

| # | Type | Required |
|---|------|----------|
| 0 | `number` | yes |

## Returns

`bool`

## Examples

### Check minimum order quantity

**Input**

```
qty = 1
```

**Template**

```
{{ qty | gte:1 }}
```

**Output**

```
true
```

### Qualify for an A grade

**Input**

```
score = 90
```

**Template**

```
{{ score | gte:90 }}
```

**Output**

```
true
```
