## Boolean

| Modifier | Description |
|----------|-------------|
| [`eq`](./eq) | Eq returns true if the value equals the given parameter. |
| [`gt`](./gt) | Gt returns true if the numeric value is greater than the threshold. |
| [`gte`](./gte) | Gte returns true if the numeric value is greater than or equal to the threshold. |
| [`not`](./not) | Not inverts the truthiness of the value. |
